POST

/api/v1/scan

Analyze a website URL for UX, performance, and accessibility issues.

Request

curl -X POST \ https://uxtester.app/api/v1/scan \ -H "x-api-key: YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{ "url": "https://example.com" }'

Response

{ "score": 85, "categories": { "performance": 92, "accessibility": 78 }, "strengths": [...], "weaknesses": [...] }
